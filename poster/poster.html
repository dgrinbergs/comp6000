SmartGen

Procedural building generation for Minecraft

Project Outline

Designing and building structures in Minecraft can be time-consuming, especially for large scale builds. We set out on adventure to simplify this process and let AI do the hard work for you!

How Does it Work?

  • Player tells the system what type of building they want
  • Using many hours of training, the AI generates several builds
